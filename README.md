# bulbie

**Bulbie** is a macOS menu bar utility that unlocks XDR brightness on Apple 
displays, allowing users to boost beyond standard SDR limits or dim below 
system minimums.

Designed for Apple XDR displays. Results may vary on other hardware.

Developed and maintained by [Pinpoint PM, LLC](https://pinpoint-pm.com)

---

## System Requirements

- macOS 12.0 (Monterey) or later
- Apple Silicon or Intel Mac with Metal support
- Best experienced on Apple XDR-capable displays

---

## Privacy Policy

**Last updated: [RELEASE DATE]**

Bulbie does not collect, transmit, store, or share any personal data or 
usage information of any kind.

Specifically:

- Bulbie does not collect personal information
- Bulbie does not use analytics or tracking of any kind
- Bulbie does not make network requests
- Bulbie does not share data with third parties
- All user preferences are stored exclusively in local `UserDefaults` on the 
user's device and never leave that device

Bulbie's only system interactions are reading display hardware capabilities 
via macOS APIs and writing brightness adjustments to the local display.

For questions regarding this privacy policy, contact: holden@pinpoint-pm.com
